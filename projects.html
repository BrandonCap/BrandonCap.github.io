Collatz Conjecture Visualizer

Technologies: Python, matplotlib JavaScript, Chart.js, HTML5, CSS3

An interactive visualization tool written in Python (rewritten in JS for the webpage demo) that demonstrates the Collatz Conjecture (also known as the 3n+1 problem). Enter any positive integer and watch as the sequence unfolds graphically, showing the fascinating pattern of numbers as they eventually reach 1. Features dynamic charting and real-time sequence generation.

Levenshtein Edit Distance Calculator

Technologies: Python, JavaScript, HTML5, CSS3

A tool written in Python (rewritten in JS for the webpage demo) that calculates the minimum number of single-character edits (insertions, deletions, or substitutions) needed to transform one string into another. This algorithm is widely used in spell-checking, DNA sequence analysis, and natural language processing. Features step-by-step visualization of the dynamic programming matrix.
