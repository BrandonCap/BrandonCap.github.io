What is the Collatz Conjecture?

The Collatz Conjecture is one of mathematics' most famous unsolved problems. Starting with any positive integer:

  • If the number is even, divide it by 2
  • If the number is odd, multiply it by 3 and add 1
  • Repeat this process with the result

The conjecture states that no matter which positive integer you start with, you will always eventually reach 1. While this has been verified for countless numbers, it has never been mathematically proven for all numbers!
